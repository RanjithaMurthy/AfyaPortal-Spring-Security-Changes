Ecosystem Adaptation Methodology For Care Providers :

Self-Learning + Assesment + Certification

Home Afya Services Ecosystem Adaptation

Afya's smart care ecosystem, being an innovative initiative, definitely calls for an extensive understanding of the program that details every aspect of the ecosystem to each stakeholder - its features, procedures, mode of use and benefits. Every member is inducted into Afya's ecosystem through this self learning management system. The program also includes continuing self learning sessions that comprises walk-throughs of the new features introduced in the ecosystem on a regular basis.

The adaptation methodology is member specific and caters to each type of stakeholder joining the ecosystem. The modules have been created by considering the self learning strategy that needs to be applied for each type of stakeholder - care provider versus payer versus patient.

The stakeholders are taken through a simulation of the real-time experience they would have within the ecosystem, which aids them in adapting to the ecosystem easily and effectively. The modules include self learning and assessment tools, which are by far one of the most effective self learning strategies. This program not only helps them appreciate the benefits of Afya's ecosystem but also gives them an overview of how Afya's processes are integrated within the ecosystem. This training program ensures that the stakeholders take advantage of all the features that the ecosystem has to offer.

We understand that self learning is a continuing process and that as a member you would definitely have queries as you use the live system. Afya's adaptation methodology also includes features that address your immediate questions and inquiries.


Overall Approach

The main feature of this adaptation methodology is that it is a comprehensive structured self learning module executed through short and continuous training sessions without affecting the balance between the time spent on routine work and that taken up for self training. The self learning program includes three phases - self learning, assessment and certification.

Self learning

  • The Self learning programs are specific to user types and are detailed.
  • These programs use scenarios to explain each administrative and functional activity within the ecosystem (eg, setting up master information, tracking clinic visits, using dashboards, building network, etc.). -
  • The purpose of each activity and its importance in simplifying the user's experience will also be detailed.
  • The duration and quality of each session is monitored - simplicity and crispness are key.
  • These sessions will encourage a steep self learning curve within a limited time frame.

Assessment

  • Self learning programs are effective only when the learner's level of understanding is assessed. The programs include simple and interesting assessments at the end of each self learning session - that is, for every session explaining each activity.
  • The assessment includes a theory section and a practical section.
  • The theory section includes objective-type questions such as multiple choice questions (MCQs), match the following and fill in the blanks, while the practical section is scenario based.

Certification

  • At the end of the self learning program, the members will be duly presented with a certificate that designates them as a ‘Certified Afya User’. This certification will signify the culmination of the online self learning process through Afya’s self learning management system.
